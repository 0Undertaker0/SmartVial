# Fase 2 — Wireframes (baja fidelidad)

Proporciono wireframes básicos por pantalla. Para ver mockups de alta fidelidad revisar `frontend/`.

1) Login
- Campo: usuario / contraseña
- Botón: Iniciar sesión

2) Dashboard (Supervisor)
- Topbar: búsqueda, perfil
- Left nav: Dashboard, Incidentes, Mapa, Usuarios, Reportes
- Main: KPIs (cards), gráfico de incidentes últimos 30 días, lista reciente

3) Crear incidente (Agente móvil)
- Formulario: Fecha/Hora, Tipo, Severidad, Coordenadas (auto GPS), Dirección, Descripción, Fotos (upload), Testigos, Comentarios
- Botones: Guardar borrador, Enviar y generar acta

4) Mapa SIG
- Controles: capas (mapa base, calor), filtros por fecha y tipo
- Marcadores con resumen en tooltip
