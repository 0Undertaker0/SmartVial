# Fase 2 — Design System (SMARTVIAL)

Nota: Mantener la paleta institucional existente. Las variables están definidas para ser ajustadas al estilo oficial.

## Paleta de colores (variables CSS)
- --sv-primary: #0b5ed7; /* Azul institucional */
- --sv-secondary: #0a76d0;
- --sv-accent: #28a745; /* Verde para éxito */
- --sv-warning: #ffc107;
- --sv-danger: #dc3545;
- --sv-bg: #f8f9fa;
- --sv-surface: #ffffff;

## Tipografías
- Primaria: Inter, system-ui, -apple-system, "Segoe UI", Roboto, "Helvetica Neue", Arial
- Pesos: 400, 500, 700

## Espaciado y grid
- Grid base: 12 columnas, gutter 24px
- Spacing scale: 4, 8, 12, 16, 24, 32, 48, 64

## Componentes reutilizables
- Button (primary, secondary, ghost)
- Input / Select / Textarea
- Card
- Modal
- Table responsive
- Data table con paginación y filtros
- Forms con validación inline
- Toast / Notification
- Avatar / User badge

## Iconografía
- Usar íconos SVG (Heroicons o FontAwesome como fallback)

## Estados visuales
- Hover, Active, Focus, Disabled, Loading

## Accesibilidad
- Contraste mínimo 4.5:1 para texto normal
- Labels visibles, placeholders no como reemplazo de label
- Navegación por teclado y lectores de pantalla
